**Bridging the Rural-Urban Teaching Gap with AI**

This platform empowers teachers and students by providing AI-assisted teaching and learning tools. Using Retrieval-Augmented Generation (RAG) and modern AI technologies, it ensures learning continuity, resource sharing, and effective doubt resolution.

---

## Features

1. **Syllabus Storage & Retrieval**  
   - Store the complete syllabus in a database.  
   - Retrieve syllabus content on-demand using RAG to answer student queries.

2. **Automated Notes & Question Generation**  
   - Generate concise lecture notes for students.  
   - Create exam-style questions based on syllabus content.  

3. **Doubt Resolution**  
   - Solve student doubts using the syllabus offline.  
   - If a query is outside the syllabus, access the internet to provide answers.

4. **Dual Chat System**  
   - **Teacher-AI Chat:** Teachers can collaborate with AI to add, update, or correct information.  
   - **Student-AI Chat:** Students can ask questions privately to AI without involving teachers.

5. **Content Filtering**  
   - Automatic detection of inappropriate content (e.g., pornography, offensive language) before answering queries.

6. **Video Upload & Note Assistance**  
   - Teachers can upload videos.  
   - AI generates notes and highlights important points from the uploaded videos.

7. **Syllabus Editing**  
   - Teachers can modify and update syllabus content directly in the platform.

8. **Reference Books Integration**  
   - Include key textbooks such as RD Sharma and JD Verma for reference and question generation.
